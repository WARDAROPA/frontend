Para poder ejecutar la aplicacion web primero hemos de clonar el repositorio.
Una vez clonado, entramos dentro de la carpeta que se nos crea y ejecutamos los siguientes comandos por terminal:

npm install

npm start

Con esto el frontend se abrira en una pestaña nueva y podremos utilizar la aplicacoion.
Importante ejecutar el backend primero para evitar errores.
